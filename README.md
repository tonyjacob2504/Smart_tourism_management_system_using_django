# 🌍 Smart Tourism Management System

## 📌 Overview

The Smart Tourism Management System is a web-based application developed to simplify travel planning and tourism management. It provides an integrated platform for users, travel agencies, hotels, and administrators with advanced features like weather prediction and price prediction.

---

## 🎯 Objectives

* Provide a centralized tourism management platform
* Help users plan trips efficiently
* Integrate weather forecasting
* Implement price prediction for budget planning
* Improve coordination between users, agencies, and hotels

---

## 👥 Modules

### 🔑 Admin

* Manage users, agencies, and hotels
* Monitor system activities
* Approve/reject service providers
* Manage destinations and packages

### 🏢 Agency

* Create and manage tour packages
* Handle bookings
* Update pricing and availability

### 👤 User

* Register and login
* Search and book packages
* View weather predictions
* Use price prediction tools
* Provide feedback and ratings

### 🏨 Hotel

* Manage hotel listings
* Update room availability
* Handle bookings

---

## 🚀 Features

* 🌦️ Weather Prediction
* 💰 Price Prediction

* 📅 Booking Management

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/smart-tourism-management.git

# Go to project folder
cd smart-tourism-management

# Create virtual environment
python -m venv env

# Activate environment (Windows)
env\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Apply migrations
python manage.py migrate

# Create superuser
python manage.py createsuperuser

# Run server
python manage.py runserver
```

1. User registers and logs in
2. Searches for destinations/packages
3. Gets weather and price predictions

5. Agencies and hotels manage bookings
6. Admin monitors the system
